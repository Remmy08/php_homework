# Стена сообщений (Вариант 0)

Домашняя работа по дисциплине "Программирование на PHP"

## Функционал
- Регистрация и авторизация пользователей
- Создание сообщений на общей стене
- Редактирование и удаление только своих сообщений
- Запрет редактирования/удаления по истечении 1 часа после публикации
- Защита от XSS, пароль хешируется через password_hash
- Чистая архитектура, шаблоны, PDO
